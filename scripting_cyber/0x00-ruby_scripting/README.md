# Ruby scripting — cyber

Petit repo pour le module Ruby cyber à Holberton. On bosse des scripts qui parlent réseau, JSON, HTTP et un peu de crypto basique. Perso je trouve ça cool parce que Ruby reste hyper présent côté pentest (Metasploit tout ça), donc autant être à l’aise vite fait.

Fichiers principaux dans ce dossier : exercices 0 à 11, plus `file.json` / `file2.json` pour la partie lecture / merge JSON.
